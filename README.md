## 它到底能帮你做什么？

### 多文档标签化管理

这是最核心的功能。装完之后，你在 Excel 里开十个表格，任务栏只有一个 Excel 图标，所有文件都作为标签显示在窗口顶部，鼠标点一下就切换，比 `Alt+Tab` 快多了。

### 右键菜单批量操作

在标签上右键，可以直接批量保存所有文件、批量关闭、重命名。以前要一个个点"保存"的步骤，现在一键搞定。对于需要同时处理大量文件的财务、数据分析岗位来说，这个功能省时间相当明显。

### 收藏夹分组一键批量打开

有个功能很多人用了才说"原来还有这个"：收藏夹。你可以把经常一起用的文件归到同一个组，比如"月报相关"、"客户资料"。早上到公司，右键收藏夹 → 选组 → 一键全开，直接进入工作状态，不用再一个个翻文件夹找。

### 不关文件直接重命名

这个细节用过的人都会觉得"这才叫用心"。以前改文件名要先关掉文档，或者打开"另存为"，现在直接在标签上操作，文件一直开着，完全不打断工作节奏。

### 自定义标签栏外观

标签位置可以调（顶部、底部、左侧、右侧），颜色、字体、形状都能改，强迫症患者和有个性化需求的用户可以捣鼓个把小时——但默认样式其实也足够好看，大多数人安装完不用动任何设置。
